➡️ Computational

Data Management

  • Frictionless Data Package (📍✨): Creates data package (meta schema) of files.
  • Goodtables.io (📍✨): Checks formatting of data. Connects to github. No coding needed.
  • Open Refine (✨): Helps to clean, format data.
  • WTFcsv (🧭✨): Can be used for first-glance into dataset.
  • SameDiff (🧭✨): Compares text files.

Social Network Analysis

  • Connect the Dots (✨🧭📍): Basic network analysis. No coding needed.
  • Social Network Visualizer (✨🧭📍): User-friendly social network analysis.
  • Cytoscape (✨): Adds attribute data.
  • Statnet (✨🥨): R packages.
  • Gephi (✨): Lots of associated packages. Runs of desktop.

Resources

  • The Turing Way: Open source guide to collaborative data science. Very comprehensive.
  • Social Network Analysis for Anthropologists (📍): Resources from workshop (with course material, additional resources, etc).
  • Introduction to R for Social Scientists: Course material.
  • Databasic.io: Suite of tools to introduce people to working with data.
  • Datajournalism.tools: Website that collects resources for data journalists.
  • How to prepare your data for analysis and charting in Excel & Google Sheets: Written by Lisa Charlotte Rost.
  • Code for Science & Society: US non-profit supporting open collaboration.
  • Tutorials & Guides: By Miriam Posner. Tutorials on data cleaning, manipulation, data visualization, mapping, network analysis, text analysis, and web development.
  • TaPoR: Digital humanities tools for text analysis and retrieval.
