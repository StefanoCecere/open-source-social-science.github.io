Open Source Tools for Social Science Researchers

Tools for social scientists of all kinds, conducting research with people online.

Want to contribute? Join the project on GitHub or request to edit on Google Docs.

About this project

While nothing beats pen & paper, this collection is for anyone who uses digital tools for their research. Drawing from the communities of web developers, data journalists, design researchers, market researchers, and UX/UI designers who have created tools for their own work, it is curated for research purposes (both qualitative and quantiative). Many of the tools listed are open-source (though some are not). But what is open source, you ask?

Open source software is a type of computer software in which source code is released under a license in which the copyright holder grants users the rights to use, study, change, and distribute the software to anyone and for any purpose. Open-source software may be developed in a collaborative public manner. (source)

While not all tools listed here follow this definition, this collection is itself remains an open source resource. Much like open source software, social science has always been collaborative, though not necessarily done in public. Its relationship with free or open access publishing is obviously more complicated. However, this project was done ____.

Others might require knowledge of more technical software or programming languages like R (learn it here or here) or Python (here or here).
